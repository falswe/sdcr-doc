◆ createTask()

mtc::Task MTCActionServer::createTask ( )
private

creates the task that the robot will need to execute

Note
this is done by concatenating multiple atomic tasks (such as pick obj...). The atomic tasks are the following:
  • open hand
  • move to pick the object
  • pick the object (close hand)
  • lift it from the table
  • move to goal position
  • open hand (handover)
  • retreat
  • return home (starting position)
